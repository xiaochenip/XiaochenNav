关于小辰导航

小辰导航是一个简洁清爽、实用高效的个人网址导航网站，旨在为用户提供分类清晰、无广告、极速访问的网址集合。

项目愿景

我们希望通过小辰导航，帮助用户快速找到所需的网站，提高上网效率，减少信息过载带来的困扰。

无论是日常工作、学习研究，还是娱乐休闲，小辰导航都能为您提供便捷的网址访问体验。

核心功能

  • 分类导航：9个精心划分的分类，涵盖常用网站、AI工具、实用工具等
  • 二级分类：支持子分类功能，更细致地组织网站内容
  • 智能搜索：支持站内搜索和全网搜索，快速找到所需网站
  • 主题切换：支持明暗两种主题，自动保存用户偏好
  • 点击排行：实时统计网站点击量，展示日榜、周榜、月榜
  • 自动获取图标：自动为网站生成favicon图标，提升视觉体验
  • 自定义网址：支持用户添加和管理个人常用网址

技术实现

小辰导航采用现代前端技术栈构建：

  • 前端框架：HTML5 + TailwindCSS
  • JavaScript：原生JavaScript，数据驱动视图
  • 数据管理：本地存储（localStorage）用于主题偏好和点击统计
  • 图标服务：自动获取网站favicon
  • 响应式设计：适配移动端和桌面端

本地部署

小辰导航支持本地部署，无需服务器，安全可靠： 

git clone https://gitee.com/xiaochenip/xiaochen-nav.git
cd xiaonav
# 直接在浏览器中打开 index.html 文件
# 或使用本地服务器
python -m http.server 8080
# 访问 http://localhost:8080
