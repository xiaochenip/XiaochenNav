小辰导航（XiaoNav）

1\. 技术选型与初始化

核心框架：HTML + TailwindCSS。

数据与逻辑：所有网站数据（如链接、描述、图标）都集中存放在JavaScript数组或一个单独的 data.js 文件中。然后用一个render()函数来动态渲染分类和网站卡片，这样后期修改数据会很方便。

&#x20;核心功能实现

利用TailwindCSS快速构建响应式界面，并通过JavaScript实现所有交互，实现一个“数据驱动视图”的架构。

分类导航：根据数据动态生成分类标签，点击时筛选下方卡片。

网站卡片：每个卡片包含图标、标题、描述，并有一个访问按钮。

实时搜索：监听输入框事件，实时根据标题或描述对卡片进行模糊搜索。

主题切换：实现明暗主题切换，并利用localStorage保存用户偏好。
